RankRoom Embeddable Widget Demo

See how easy it is to embed decision rooms in your application

Basic Widget

Create and manage decision rooms with minimal setup

<div id="rankroom-widget-1"></div>
<script>
  const widget = RankRoomWidget.create({
    containerId: 'rankroom-widget-1',
    serverUrl: 'http://localhost:3000'
  });
</script>

Pre-configured Room

Start with a specific room configuration

<div id="rankroom-widget-2"></div>
<script>
  const config = {
    room: {
      title: "Team Lunch Ideas",
      votingSystem: "dot-voting",
      votesPerParticipant: 3
    }
  };
  RankRoomWidget.fromConfig(config, 'rankroom-widget-2');
</script>

MCP Server Integration

Use with AI assistants and automation tools

// AI can create and manage decision rooms
const roomResult = await mcpServer.call('create_decision_room', {
  title: 'Project Ideas for Q2',
  votingSystem: 'alternative-voting'
});

// Monitor and summarize results
const summary = await mcpServer.call('get_room_summary', {
  roomId: roomResult.roomId
});

console.log(summary); 
// "Decision room 'Project Ideas for Q2' had 8 ideas from 5 participants. 
//  Top idea: 'Mobile app redesign' with 12 votes."
